# NeuroCare AI

NeuroCare AI is an AI-powered web application that helps parents understand child developmental milestones through AI-generated educational guidance, interactive assessments, and an AI chatbot.

## Features

- AI-powered developmental assessment
- Personalized educational reports
- AI Chat Assistant
- Learn section
- About page
- Responsive UI

## Tech Stack

- React.js
- Vite
- Node.js
- Express.js
- Tailwind CSS
- Google Gemini API
- Render
- Vercel

## Installation

### Clone the repository

```bash
git clone https://github.com/adithya-adari/NeuroCare-AI.git
```

### Install dependencies

Frontend:

```bash
cd client
npm install
npm run dev
```

Backend:

```bash
cd server
npm install
npm run dev
```
